
# Product Overview

Forage is not a GitHub dashboard.

It is a repository discovery and rediscovery tool focused on turning years of accumulated GitHub stars into useful insights.

Primary workflow:

Landing
→ GitHub Login
→ Import Stars
→ Normalize Data
→ Analyze
→ Explore Dashboard
→ Export Results

MVP focus:
- Useful categorization
- Interesting insights
- Strong filtering
- Explainable rankings
- Fast local experience

MVP user promise:
Connect GitHub, import starred repositories into the browser, generate local analysis, and export results without Forage storing the user's repository data on the server.
