
# Forage Executive Summary

Forage is a privacy-first GitHub repository rediscovery platform.

Primary goal:
Help developers rediscover useful repositories buried in years of GitHub stars.

Core principles:
- GitHub Auth Only
- Local First
- PWA First
- No AI in MVP
- Minimal User Account
- Explainable Scoring
- Config-Driven Categorization
- Browser-Based Analysis

Minimal User Account means Forage stores only the data required to connect to GitHub and respect user settings:
- GitHub auth token material or token references
- A non-obvious GitHub user hash
- Opt-in or opt-out preferences
- Small application settings

Forage does not store repository lists, generated analysis, reports, exports, or other user repository data on the server.
